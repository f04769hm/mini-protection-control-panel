# Project Definition

This is a low-voltage functional demonstrator of a substation protection and control workflow.

It models a simple 33/11 kV system with a downstream feeder relay and upstream backup relay, covering fault-current calculation, overcurrent settings, IDMT grading, software FAT-style testing, a physical trip-circuit, watchdog/healthy supervision and panel documentation.

The hardware is not a protection-class relay, CT measurement chain or certified IEC 61850 IED — it is a safe extra-low-voltage demonstrator built to practise the engineering workflow.
